# Traction
Open source Electric traction drive systems for AgTech

## Tractor.mmd ##
High level diagram of the Solar Tractor control.

Main components are as follows:

1. Here4 with rover flight control firmware build (https://docs.cubepilot.org/user-guides/here-4/here-4-manual#id-8.-installing-ardupilot-on-here4)
2. OpenBuilds Blox controller (2 per machine), (https://www.youtube.com/watch?v=3IvTljqS-3s )
3. Vesc controller (2 per machine), (https://github.com/tmagik/bldc)
